Based on the visual analysis of the provided video, here is a detailed breakdown of the GALAXY TUNNEL Android app's UI/UX:

### 1. Navigation
*   **Sidebar Menu:** The app utilizes a classic hamburger menu in the top-left corner. This sidebar contains primary navigation links: "Servers," "Settings," and "About Us" (represented by icons).
*   **Top Bar:** Features the app name, a subtitle, and a language toggle (MY/EN) in the top-right corner for quick switching.
*   **No Bottom Bar:** The app opts for a clean, single-page main interface rather than a bottom navigation bar, keeping the focus on the connection status.

### 2. Main Screen Layout
*   **Connection Engine:** The centerpiece is a large, circular power toggle button. 
    *   **Off State:** Shows a standard power icon.
    *   **On State:** Transforms into a green shield icon with a subtle pulse animation, indicating an active, secure connection.
*   **Status & Metrics:** Once connected, the app displays real-time data:
    *   **Timer:** Shows the duration of the current session.
    *   **Speed Metrics:** Displays Download (DL) and Upload (UL) speeds in MB/s, accompanied by upward and downward arrow icons.
*   **Server Selection:** Directly below the connection button, the currently selected server is displayed. A collapsible "Server List" section allows users to expand or hide the available nodes.

### 3. Server List Appearance
*   **List View:** Servers are presented in a vertical list with radio buttons for selection.
*   **Latency Indicators:** Each server entry includes a real-time latency (ping) value in milliseconds (ms), color-coded in green (low latency) for easy identification of the fastest nodes.
*   **Quick Actions:** The server list header includes a "Refresh" icon to update pings and a "Link" icon to add custom configurations.

### 4. Color Scheme and Animations
*   **Themes:** The app supports both **Dark Mode** (default, using deep greys and blacks) and **Light Mode** (white background with dark text).
*   **Accent Colors:** Green is used consistently to signify "Active," "Connected," or "Good Latency." Orange is used during the "Connecting" transition.
*   **Animations:** 
    *   The sidebar has a smooth slide-in transition.
    *   The power button features a sliding toggle animation and a rhythmic pulse effect when the VPN is active.
    *   Collapsible sections expand and retract with fluid motion.

### 5. Unique Features
*   **Custom Config Support:** A dedicated button allows users to manually paste `vless://` or `trojan://` configuration links, providing flexibility for advanced users.
*   **Granular Settings:** The settings menu offers:
    *   **Text Size Adjustment:** Users can choose between Small, Medium, and Large text for better accessibility.
    *   **Diagnostics & Session Logs:** A technical section for troubleshooting and viewing connection history.
*   **Integrated Sharing:** A "Share App with Friends" button is prominently placed within the settings for easy referral.
*   **System Integration:** The app provides a persistent notification with a "Disconnect" shortcut and real-time status, allowing control without reopening the app.
